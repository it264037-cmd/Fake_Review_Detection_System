ReviewGuard Setup
Configure EmailJS for real OTP verification, then connect to the Python ML backend.
1
Start the Python ML Backend
In a terminal, navigate to the backend/ folder and run:

pip install -r requirements.txt python train_model.py "C:\Users\D.praveen\Downloads\train_8000.csv" uvicorn app:app --reload --port 8000
The API will be live at http://localhost:8000 and will use your train_8000.csv spam-review dataset.
Swagger docs: http://localhost:8000/docs
2
Create free EmailJS account for OTP emails
Sign up at emailjs.com (free, 200 emails/month). Add Gmail service, create a template with {{otp_code}} variable, and copy your 3 keys below. Open emailjs.com
Template body must include: {{otp_code}}, {{to_email}}, {{to_name}}
